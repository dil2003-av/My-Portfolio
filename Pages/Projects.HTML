SUN HOUSE RESTAURANT MANAGMENT SYSTEM

I developed the Sun House Restaurant System using Java, incorporating object-oriented programming concepts to ensure scalability and maintainability. The user interface was crafted with JavaFX and Scene Builder, providing an intuitive and user-friendly design. The system integrates a MySQL database for efficient data management, utilizing JDBC for seamless database connectivity. This project streamlines restaurant operations, including managing orders, reservations, and inventory, showcasing my expertise in Java programming, OOP principles, database integration, and UI/UX development.
